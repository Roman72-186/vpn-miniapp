# VPN Mini App

Миниапка для платной выдачи VPN-конфигов через отдельный VPN-сервер.

## Схема

- `GitHub` хранит код miniapp.
- `Vercel` публикует фронт по HTTPS.
- `n8n` принимает запросы от miniapp, создаёт оплату, проверяет статус и выдаёт доступ.
- Новый сервер `72.56.28.245` хранит `AmneziaWG` и API выдачи конфигов.

## Публичный URL

- Production: `https://vpn-miniapp-pi.vercel.app`
- Репозиторий подключён к `Vercel`, поэтому новые push в `main` будут деплоиться автоматически.

## Поток работы

1. Пользователь открывает miniapp по ссылке из Telegram.
2. Miniapp отправляет `initData` в `n8n`.
3. `n8n` создаёт счёт на 30 дней.
4. После оплаты `n8n` вызывает VPN API на новом сервере.
5. VPN API создаёт конфиг и сохраняет срок действия.
6. Miniapp показывает пользователю ссылки на приложения и его `.conf`.

## Что важно

- Основной старый VPN не используется для новых платных доступов.
- Во фронте нет секретов.
- Все ключи и выдача конфигов живут только на новом VPN-сервере.

## Следующие шаги

1. Заполнить URL webhook'ов в `config.js`.
2. Запушить проект в GitHub.
3. Подключить репозиторий к Vercel.
4. Поднять VPN API и `AmneziaWG` на новом сервере.
